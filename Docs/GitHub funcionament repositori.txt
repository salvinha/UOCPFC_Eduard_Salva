El primer que hauríeu de fer és crear un projecte en github comú per a tots dos i fer una estructura bàsica per al projecte, tipus:

Docs
Memoria
   --> memoria_eduard
   --> memoria_salvador
Implementacio
   --> client
   --> server

Haurieu de fer un projecte en comú compartit per a 2 usuaris (vosaltres).  Haurieu de tenir com a mínim 4 branques:

producció
   --> dev
         ---> eduard
         ---> salvador

La branca de producció (master) en general es deixa per a versions finals.  De la versió master es penja una versió dev on hi ha tots els merges de les versions pròpies de cadascú.  Cadascú té una branca pròpia.  Cada cop que funciona alguna cosa i es dona per fet, llavors es col.loca en la branca de dev.  De tant en tant, es comprova que tot funciona i es puja  a la branca de producció.

Tot plegat és independent de la tecnologia d'implementació i del tipus de servidor.  Llegiu-vos el tema sobre el github que tinc en el forum penjat.

Quan tingueu solucionat el github, llavors crec que ja tindrem un servidor per a que pugueu pujar codi.
